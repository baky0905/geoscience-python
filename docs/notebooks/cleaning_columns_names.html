Clean Column Names¶

Replace something with something - str_replace¶

First we List columns names and we spot that there are parts of the column names we would like to get rid of - for the time being.

We need to clean them so we make the subsequent workflows on dataframes as convenient as it can get.

Task:

Remove wlb, fld, fcl prefixes.

It is important that we wrap our chains of functions into parantheses. Alternative approach would be to end each line with a backslash.

I find the first approach more elegant so I will use it.

Clean column names with clean_names method (janitor module)¶

Method clean_names (janitor module):

  • takes all column names

  • converts them to lowercase

  • replaces all spaces with underscores

This method does not mutate the original DataFrame.

[8]:

import janitor

[9]:

wellbore_exploration_all = (
    wellbore_exploration_all.clean_names(case_type="snake")
)

Rename multiple columns by predefining a dictionary.¶

[13]:

new_column_names = {
    "drilling_operator" : "DRILLING_COMPANY",
    "purpose" : "WELL_PURPOSE",
    "total_depth" : "TOTAL_DEPTH_METERS"
}

(wellbore_exploration_all
   .filter(items=["drilling_operator", "purpose", "total_depth"])
   .rename(columns=new_column_names) ##  "old_name":"new_name"
)
